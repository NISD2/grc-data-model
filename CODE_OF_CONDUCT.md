# Code of conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). In short:

- Be kind. Assume good intent. Disagree with substance.
- Domain disagreement is healthy — regulatory interpretation is genuinely hard. Cite sources.
- No harassment, discrimination, or personal attacks.

Issues with conduct: **contact@nisd2.eu**.
